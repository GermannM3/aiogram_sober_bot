# Sober Bot

Бот для мониторинга чата на предмет трезвости участников.

## Запуск локально

1. Установите зависимости:
   ```bash
   pip install -r requirements.txt


2. Запустите бота:

   python main.py


Запуск в Docker


docker build -t sober-bot .


docker run -e BOT_TOKEN=YOUR_BOT_TOKEN sober-bot
